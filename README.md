# Load Test App

Сервис на NestJS + фронт на Vue 3 + нагрузочное тестирование.

## Запуск проекта

```bash
docker-compose up --build
```

## URL

- **Frontend**: http://localhost:8080
- **Backend API**: http://localhost:3000

## Оптимизации

Были применены следующие оптимизации для эндпоинта GET /items:

1. **Индексы в PostgreSQL** - на полях `name` и `created_at`
2. **Connection Pool** - пул соединений с БД (max: 20)
3. **In-Memory кэширование** - кэш на уровне приложения с TTL 5 сек
